# ReadAlong — Vision & Concept

## One-line statement

> 読んだ本のメモを、友達と気軽に共有できる軽い読書ノート。

## Concept

**Why** — 読書のメモは溜まるほど価値が出るが、Notion / Evernote のような汎用ツールは重く、専用 SNS は派手すぎて日常使いに馴染まない。気軽に書いて、気軽に共有できる "間のサイズ" のツールが欠けている。

**Who** — 月に 2-5 冊読む社会人。読書記録は付けたいが、感想を完璧にまとめる気力はない。SNS のフォロワーよりも、特定の友人 3-10 人と "あの本どうだった？" の温度感で交換したい。

**What** — 本を登録し、メモを書き、公開・非公開を切り替えるだけのコア。1 メモ = 1 思考。ツリー化せず、フォルダもつくらない。本に紐付くメモが時系列で並ぶだけ。

**How it's different** — Goodreads のように "評価とレビュー" に寄せず、Notion のように "整理" にも寄せない。"思いついたことを書き留めて、見せたい人にだけ見せる" が一行で完結する。

## Assumptions used

- ターゲットは個人（B2C）、当面は無料
- メモは Markdown で十分。リッチテキストは扱わない
- 友達は招待制（公開 SNS にしない）
