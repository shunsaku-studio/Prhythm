## Phase 1 quality rules

- Every entry must have **Why listed** — no directory dump
- Tag **Verified** / **二次情報** / **要確認** on scale and pricing
- Check pivot/shutdown before listing
- Search both Japanese and English when global references apply
- Run 5–10 varied searches across function / UX / user-context axes
- Check at least 2 of: Product Hunt, G2/Capterra, App Store category, STARTUP DB, INITIAL
